# DivAssist
This is DivAssist!  A webapp companion to the Divvy Bikeshare service.  
  
### How to run
Navigate to the primary project folder divassist/ (make sure you are in the same directory as Makefile)  
To start the server:  
`make run`  
To run the tests:  
`make tests`
